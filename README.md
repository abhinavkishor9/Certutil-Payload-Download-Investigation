# Certutil-Payload-Download-Investigation

## 📌 Overview

This project demonstrates how the legitimate Windows utility **certutil.exe** can be abused as a **Living Off The Land Binary (LOLBin)** to retrieve remote content.

The objective of this investigation was to analyze **Sysmon Event ID 1 (Process Creation)** generated by the execution of `certutil.exe` and identify suspicious command-line activity associated with payload retrieval techniques.

Although the download operation did not complete successfully due to local system restrictions, the process execution generated valuable telemetry that can be used for threat hunting and incident investigation.

This investigation focused on:

## 🔬 Lab Procedure

### Step 1 - Create a Test File

```cmd
mkdir C:\Lab
echo Hello from Certutil Lab > C:\Lab\test.txt
```

---

### Step 2 - Start a Local HTTP Server

A local HTTP server was started to host the test file.

The server was verified by opening:

```
http://localhost:8000/test.txt
```

The browser displayed:

```
Hello from Certutil Lab
```

confirming that the file was accessible.

---

### Step 3 - Execute Certutil

Run the following command:

```cmd
certutil -urlcache -split -f http://localhost:8000/test.txt test.txt
```

The command returned:

```
Access is denied.
```

Although the download failed, **Sysmon Event ID 1** successfully captured the process execution.

---

### Step 4 - Investigate Sysmon Logs

Open:

```
Event Viewer
└── Applications and Services Logs
    └── Microsoft
        └── Windows
            └── Sysmon
                └── Operational
```

Filter for:

```
Event ID: 1
```

Locate the execution of **certutil.exe**.

Review the following fields:

- Image
- ParentImage
- CommandLine
- User
- ProcessId
- ProcessGuid
- Timestamp

---

## 🔍 Detection Logic

Generate an alert when:

- `certutil.exe` executes
- `-urlcache` appears in the command line
- `-split` appears in the command line
- `-f` appears in the command line
- HTTP or HTTPS URLs are present

## 🚨 Severity

**Overall Severity:** High

The attempted use of **certutil.exe** to retrieve remote content is a well-known attacker technique and should be investigated regardless of whether the download succeeds.

---

## ⚠️ False Positives

- Certificate management
- File hashing
- PKI administration
- Internal administrative scripts
- Software verification

---

## 🛑 Recommended Containment

Terminate suspicious `certutil.exe` processes, investigate the parent process and command-line arguments, and isolate the affected endpoint if malicious activity is suspected.

---

## 📌 Conclusion

This investigation demonstrates how **certutil.exe** can be abused as a **Living Off The Land Binary (LOLBin)** for payload retrieval attempts.

Although the payload download failed due to local system restrictions, **Sysmon Event ID 1** successfully captured the execution of `certutil.exe` and its associated command-line arguments, providing valuable telemetry for detection and investigation.

From a SOC perspective, **attempted LOLBin abuse is itself a valuable detection opportunity**, as it reveals attacker intent and supports threat hunting even when the intended action does not succeed.
